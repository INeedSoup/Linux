# Debian Branches
A key feature of Debian is its use of different release channels, known as Debian branches. Users can chooses the that best fits their needs for stability versus new features.
- Stable: This is the official release. It prioritises reliability and security over having the latest software versions 
- Testing: This branch contains packages that are being prepared for the next Stable release
- Unstable: Also known as "Sid", this is where active development happens. It has the newest packages but is also the most likely to have issues.
Testing and Unstable are considered rolling releases, This means they receive continuous updates.

# Package Management
Debian uses apt (Advanced Package Tool), to install, upgrade, and manage software.
